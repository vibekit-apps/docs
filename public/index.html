Build and deploy full-stack apps with AI. One API call.

Quickstart

Get your API key and deploy your first app in under 2 minutes.

1. Get an API Key

curl -X POST https://vibekit.bot/api/v1/auth/register \
  -H "Content-Type: application/json" \
  -d '{"email": "you@example.com"}'

2. Submit a Task

curl -X POST https://vibekit.bot/api/v1/task \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{"prompt": "Build a todo app with dark mode"}'

3. Check Status

curl https://vibekit.bot/api/v1/task/TASK_ID \
  -H "Authorization: Bearer YOUR_API_KEY"

Authentication

All API requests require a Bearer token:

Authorization: Bearer vk_your_api_key

Register (Get API Key)

POST /api/v1/auth/register
Content-Type: application/json

{"email": "you@example.com"}

Returns your API key and $0.25 free credits.

Tasks

Submit Task

POST /api/v1/task
Authorization: Bearer YOUR_API_KEY
Content-Type: application/json

{
  "prompt": "Build a landing page for a SaaS product",
  "skills": ["solana"],
  "projectId": "abc123",
  "callbackUrl": "https://..."
}

Get Task Status

GET /api/v1/task/:taskId
{
  "taskId": "task_abc123",
  "status": "complete",
  "deployUrl": "https://...",
  "repoUrl": "https://github.com/...",
  "projectId": "abc123"
}

Webhooks

Get notified when tasks complete. Include callbackUrl in your task.

{
  "taskId": "task_abc123",
  "status": "complete",
  "deployUrl": "https://...",
  "repoUrl": "https://github.com/..."
}

Verify signature with X-VibeKit-Signature header (HMAC-SHA256 of body using API key).

CLI

npm install -g vibekit-cli
vibekit login
vibekit build "Create a blog with markdown support"
vibekit status task_abc123

MCP Server

npm install -g vibekit-mcp

Add to Claude Desktop config:

{
  "mcpServers": {
    "vibekit": {
      "command": "vibekit-mcp",
      "env": {"VIBEKIT_API_KEY": "vk_your_key"}
    }
  }
}
